## üé® Demo Best Practices

### Before the Demo

1. **Prepare Test Sites**:
   - Wikipedia article (for research demo)
   - Amazon product page (for shopping demo)
   - YouTube video (for learning demo)

2. **Pre-load Whispers**:
   - Create 5-10 whispers with varied content
   - Include different colors and transcripts
   - Test search functionality

3. **Check Technical Setup**:
   - Internet connection stable
   - Microphone working
   - Extension loaded and updated
   - API key validated
   - Browser volume at 80%

4. **Backup Plan**:
   - Record a video demo
   - Have screenshots ready
   - Prepare offline slides
   - Export sample JSON

5. **Watch Time**: Keep to 3-5 minutes max

1. **Answer Questions Confidently**
2. **Provide GitHub Link** (if available)
3. **Offer to Show More** (export, settings, etc.)
4. **Thank Judges** and mention you're open to feedback
